# VolunteerHub

VolunteerHub is a Full Stack Volunteer Registration and Management System built using Flask, MySQL, Bootstrap, HTML, CSS, and JavaScript.

## Features

* Volunteer Registration
* User Login Authentication
* User Dashboard
* Admin Dashboard
* Volunteer Management
* Event Management
* MySQL Database Integration
* Responsive UI
* Session Management

## Technologies Used

* Python Flask
* MySQL
* HTML
* CSS
* Bootstrap
* JavaScript
* Git & GitHub

## How to Run

1. Install requirements
   pip install -r requirements.txt

2. Create database using database.sql

3. Run application
   python app.py

4. Open
   http://127.0.0.1:5000
